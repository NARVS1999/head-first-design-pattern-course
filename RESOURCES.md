# Head First Design Patterns Resources

## Knowledge

- [Book: _Head First Design Patterns_ (2nd Edition) — Eric Freeman & Elisabeth Robson](https://www.oreilly.com/library/view/head-first-design/9781492077992/)
  Ang primary source ng course na ito. Lahat ng lessons ay naka-base sa chapters ng book na ito. Use for: lahat ng pattern explanations, code examples, at exercises.

- [Refactoring.Guru — Design Patterns](https://refactoring.guru/design-patterns)
  Malinaw at structured na explanations ng bawat pattern, may UML diagrams. Use for: alternative explanations kapag hindi clear ang sa book.

- [Baeldung — Design Patterns in Java](https://www.baeldung.com/java-design-patterns)
  Practical Java implementations ng mga patterns. Use for: additional code examples at real-world usage.

- [SourceMaking — Design Patterns](https://sourcemaking.com/design_patterns)
  Comprehensive pattern catalog na may intent, motivation, at structure. Use for: pattern comparison at structure diagrams.

- [w3resource — Java Design Patterns](https://www.w3resource.com/java-design-pattern/)
  Beginner-friendly na tutorials with examples. Use for: quick reference ng pattern structures.

## Wisdom (Communities)

- [r/java](https://reddit.com/r/java)
  Malaking community ng Java developers. Use for: pattern discussions, implementation questions, real-world experiences.

- [r/designpatterns](https://reddit.com/r/designpatterns)
  Dedicated subreddit para sa design patterns. Use for: pattern comparisons, "when to use" discussions.

- [Stack Overflow — Design Patterns Tag](https://stackoverflow.com/questions/tagged/design-patterns)
  Q&A format na may highly-voted answers. Use for: specific implementation problems, edge cases.

- [Head First Design Patterns Study Group (Goodreads)](https://www.goodreads.com/group/show/38375-head-first-design-patterns)
  Goodreads group na nagdi-discuss ng book. Use for: chapter discussions, alternative perspectives.

## Gaps

- Walang dedicated video course na naka-sync sa Head First book structure
- Limited Taglish/Tagalog resources para sa design patterns
- Walang official errata page para sa 2nd edition
